# DigiTools Platform

## 📌 Description

DigiTools is a modern web platform where users can explore and access premium digital tools like AI content generators, design kits, and productivity software.

## 🚀 Technologies Used

* React.js
* Tailwind CSS & DaisyUI
* JavaScript (ES6+)
* React-Toastify
* JSON

## ✨ Features

* Add to cart functionality
* Remove products from cart
* Dynamic total price calculation
* Toast notifications for actions
* Clean and responsive UI

## 🔥 Optional Features

* Cart count updates in Navbar
* Button changes to "Added" after click
